  <files>src/main/kotlin/nesski/de/services/SystmAuthService.kt</files>
  <action>
Create SystmAuthService with:
1. login(username, password) function - executes GraphQL login mutation:
   - Mutation: login(username: $username, password: $password) { token, expiresAt }
   - Uses SystmGraphQLClient to POST to api.thesufferfest.com/graphql
   - Returns LoginResponse with token and expiresAt
   
2. validateJWT(token) function using java-jwt:
   - Decode JWT using JWT.decode(token)
   - Extract exp claim
   - Throw JWTVerificationException if token expired
   - Return DecodedJWT

3. Request/Response data classes:
   - LoginMutationRequest/Response
   - LoginData with token and expiresAt fields

Use java-jwt 4.5.1 per RESEARCH.md - add dependency if needed
  </action>
  <verify>Service compiles, login() returns token, validateJWT() throws on expired token</verify>
  <done>SystmAuthService with login() and validateJWT() functions</done>

1. GET /systm-login - returns HTML login form with username/password fields

3. GET /systm-logout - clears session and redirects to /systm-login

Follow existing route patterns from WahooAuthenticationOauth2.kt
