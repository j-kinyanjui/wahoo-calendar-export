- [x] **CI-01**: Gradle build runs automatically on push and PR to main

  - _Outcome: validated — `gradle/actions/setup-gradle@v4` handles caching automatically_

- [x] **CD-03**: GitHub Release is created with changelog when a version tag is pushed
  - _Outcome: validated — `softprops/action-gh-release@v2` creates release with auto-generated changelog_
